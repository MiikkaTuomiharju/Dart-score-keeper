# Darts Score Keeper

Browser-side single-page web application for keeping score in Darts games (301 up and 501 up).

## Features
- Two players
- 301 / 501 game modes
- Set and leg tracking
- Turn-based score tracking
- Real-time score updates
- Winner declaration
- Responsive and accessible UI

## Technologies
- HTML
- CSS
- JavaScript
- Git

## Notes
Application state is not stored permanently and resets on page refresh.
